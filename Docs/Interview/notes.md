# 简历注意事项
- 简历一定一定一定要用 pdf 格式
- 最好最近项目写最上面
- 技术要点突出
- 标点符号, 中英文统一
- 个人评价不要写, 既不能证实, 也不能证伪, 没有意义
- 中英文之间的空格, 要么全都有, 要么就全都没有
- 英文大小写, 规范统一
  - 全大写
  - 全小写
  - 全首字母大写
  - 官方大小写
- 注意层级关系的字体大小
- 不要加与自己不相关的头像
- 拼写错误
- 名字生僻字加拼音
- 对齐
